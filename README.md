## gitでブランチ作る
ブランチは核のこと、マークダウンは- (ハイフンスペースだよ)

- 今のブランチ確認
```
git branch
```
- ブランチの作成
```
git brunch test_branch
```

- ぶらんちの移動
```
git checkout test_branch
```
